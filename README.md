# 젬톡(GemTalk)

## Intro
각 플레이어에게 주어지는 미션카드에 맞는 보석 개수를 가장 빨리 모으는 플레이어가 승리하는 카드 게임입니다.  

## 목적
비대면으로 수업을 수강하다 대면으로 전환했을 때, 서로 많이 어색합니다.  
이러한 어색함을 조금이나마 해소하기 위해 만든 카드 게임입니다.

## 진행기간
